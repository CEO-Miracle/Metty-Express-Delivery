# Metty Express Delivery

Metty Express Delivery is a web-based shipping and tracking application built with HTML, CSS, JavaScript, and Firebase. Admins can submit shipments in the admin dashboard, which are securely saved in Firestore. Users can track their shipments using the generated tracking ID, providing a seamless and reliable delivery experience.

## Features

- Admin dashboard to submit and manage shipments  
- Shipments stored securely in Firebase Firestore  
- Users can track shipments using the tracking ID  
- Responsive and clean user interface for both desktop and mobile  
- Real-time updates for shipment status  
- Frontend integration with Firebase for authentication and database management  

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- Firebase (Authentication & Firestore Database)

## How to Use

1. Admin logs in to the dashboard and submits a shipment  
2. A tracking ID is generated and saved in Firestore  
3. Users enter the tracking ID on the website to track their shipment  
4. Shipment status updates in real-time
